DIFFERENT TYPES OF CSS SELECTORS

1.Element selector : The element selector selects HTML elements based on the element name.

2.Class selector : There can be multiple classes.
(The classes can have same names).
3.Id selector : The CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.

4.Grouping selector : The same style definition is implemented over multiple elements. Code size gets minimized.Commas are used to isolate every selector while grouping; hence it easily readable.

(A grouping selector in CSS allows you to apply the same styles to multiple elements by grouping their selectors together, separated by commas).

5.Pseudo class selector : Unvisited link.

6.THIS IS A PSEUDO-ELEMENT SELECTOR.
This is used to style the specified parts of an element.
