CSS DIFFERENT TYPES
1.External CSS
External CSS is a method of styling HTML documents by linking to a separate CSS file.It allows for separation of content (HTML structure) from presentation (CSS styles).This separation makes the codebase more organized, easier to maintain, and promotes reusability of styles across multiple web pages.Once defined, an external CSS file can be linked to multiple HTML documents.This ensures consistent styling across all pages, making updates and changes efficient.By caching the external CSS file, browsers can load styles faster, leading to improved page load times and overall performance.
2.Internal CSS
Internal CSS is useful when you want to apply specific styles to a single HTML document without affecting other documents.Internal CSS applies styles locally within a single HTML document.Styles defined internally affect only the elements within that specific HTML file.Internal CSS can override external stylesheets for elements within the same HTML document.This allows for more targeted styling adjustments.Compared to external CSS, where styles are stored in separate files, internal CSS is more convenient for smaller projects or when making quick styling adjustments to a single page.

3.Inline CSS

Inline CSS allows you to specify CSS rules for a specific element without affecting other elements on the page.Inline CSS applies styles directly to a specific HTML element.Inline CSS is straightforward to implement and allows for quick styling adjustments directly within the HTML markup.
